Detailed Description

Small wrapper around string literals to enable saving references to them in class members.

The goal of this wrapper is to be able to save a reference to a string literal within a class. The wrapper itself doesn't enforce that the saved string is actually a literal and mostly serves as a reminder for the developer that this parameters should be string literal.
